📖 尾部训看板使用说明

一、角色说明
本看板分两类使用者：
管理员（你）：负责每周上传 Excel 数据并发布到云端，需配置 GitHub Token。
培训师：打开链接后直接点"从云端加载"查看数据，无需上传，无需配置。
二、管理员每周操作流程
1. 首次使用：点右上角 ⚙ 配置，填入 GitHub Token 和用户名保存（仅需配置一次）
2. 每周一：点 上传 Excel，选择当周最新的回炉训流程表
3. 数据加载完成后，点 发布数据（橙色按钮），等待提示"发布成功"
4. 发布后所有人刷新页面即可看到最新数据
三、培训师使用流程
1. 打开看板链接，输入密码进入
2. 点 从云端加载（绿色按钮）获取最新数据
3. 在筛选器中选择自己的名字，查看专属视图
4. 专属视图显示：名下参训人数、达标占比、高风险人数、连续触发人数
四、风险预警说明
高风险 两次任务均不过关 → 建议绩效打C
中风险 任务一不过关且任务二仍在监控中
关注 训后差异率高于触发差异率（越训越差）
五、达标标准（手机岗位）
模块一：40台抽检，差异率 ≤ 7.5%
模块二：40台抽检，差异率 ≤ 6%
项目目标：当月触发人员达标占比 > 75%
六、连续触发人员
同一人（姓名+站点+组织相同）在数据中出现2次及以上，视为连续触发。
这批人是绩效干预的核心关注对象，触发3次及以上标红显示。

配置 GitHub Token

Token 仅保存在你的浏览器本地，不会上传到任何地方。只有配置了 Token 的人才能发布数据，其他人只能查看。
